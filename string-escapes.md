# String Escapes
Strings should support the following escapes sequences:

- `\a': bell
- `\b`: backspace
- `\f`: formfeed
- `\n`: newline
- `\r`: carriage return
- `\t`: horizontal tab
- `\v`: vertical tab
- `\0`: null character
- `\'`: single quote
- `\"`: double quote
- `\\`: backslash
- `\#`: hash
